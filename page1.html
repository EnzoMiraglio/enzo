Qu'est-ce que react ?

Réact est une biblioteque libre et devellopé par facebook ,son but ?
son but est de facilliter la creation d'application web monopage(application via une page web unique),React est une bibliothèque qui ne gère que l'interface de l'application, considéré comme la vue dans le modèle MVC. Elle peut ainsi être utilisée avec une autre bibliothèque ou un framework MVC comme Angular. La bibliothèque se démarque de ses concurrents par sa flexibilité et ses performances, en travaillant avec un DOM virtuel et en ne mettant à jour le rendu dans le navigateur qu'en cas de nécessité,cette bibliotheque est utilisé notamment pour netflix,yahoo,sony,aisni que par les équipes de facebook elle meme,appliquant le dogfooding qui est une expression significant a utilisé ses propres produit
